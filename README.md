# Learning plattform
This Razor page application uses **Entity Framework Code first** method. The goal with this project is to solve the problem of data loss and provide solution for faster performance. It handles school aspects such as:
* User.
* Courses.
* Schedule.
* Report.
* CRUD.
* Assignments.
* Submission.

# Technologies Used
*  C#
* .NET CORE.
*  Entity Framework Core
*  ASP NET CORE Razor Pages

# Database
* SQL Server Management Studio (SSMS) is used to store all the tables.
* Entity Framework is the orm (object relational mapping) that interact with sql server via linq syntax.
* Database Migrations is used for updating database scheme if something is forgotten or adding a new item.
